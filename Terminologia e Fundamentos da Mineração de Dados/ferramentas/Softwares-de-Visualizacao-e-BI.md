## 📈 1. Tableau

**Descrição:** Permite a criação de visualizações interativas e dashboards intuitivos. É uma das plataformas líderes de mercado em Business Intelligence e análise visual de dados, permitindo conectar-se a diversas fontes de dados e criar visualizações sofisticadas com operações drag-and-drop, sem necessidade de programação.

- **Desenvolvedor:** Tableau Software (adquirida pela Salesforce em 2019)
- **Plataforma:** Windows, macOS, Web
- **Licença:** Comercial (versão gratuita disponível — Tableau Public)
- **Site oficial:** <https://www.tableau.com/>
- **Download:** <https://www.tableau.com/products/desktop/download>
- **Tableau Public (gratuito):** <https://public.tableau.com/>
- **Versões disponíveis:**
  - **Tableau Desktop** — Ferramenta completa de análise e visualização
  - **Tableau Server** — Solução on-premises para compartilhamento e colaboração
  - **Tableau Cloud** — Versão SaaS hospedada na nuvem
  - **Tableau Public** — Versão gratuita para publicação de visualizações públicas
  - **Tableau Prep** — Ferramenta de preparação e limpeza de dados
- **Documentação:** <https://help.tableau.com/>
- **Comunidade:** <https://community.tableau.com/>
- **Tableau para estudantes:** <https://www.tableau.com/academic/students>

---

## 📊 2. Power BI

**Descrição:** Ferramenta da Microsoft para visualização de dados e Business Intelligence. Permite criar relatórios interativos e dashboards conectados a centenas de fontes de dados, com integração nativa ao ecossistema Microsoft (Excel, Azure, SQL Server, SharePoint). Oferece recursos de modelagem de dados, DAX (Data Analysis Expressions) e integração com linguagem R e Python.

- **Desenvolvedor:** Microsoft
- **Plataforma:** Windows (Desktop), Web, Mobile (iOS, Android)
- **Licença:** Freemium (versão gratuita + planos pagos Pro e Premium)
- **Site oficial:** <https://powerbi.microsoft.com/>
- **Download (Desktop):** <https://powerbi.microsoft.com/desktop/>
- **Power BI Service (Web):** <https://app.powerbi.com/>
- **Versões disponíveis:**
  - **Power BI Desktop** — Gratuito, ferramenta completa de criação de relatórios
  - **Power BI Pro** — Licença por usuário para compartilhamento e colaboração
  - **Power BI Premium** — Capacidade dedicada para grandes organizações
  - **Power BI Mobile** — Aplicativo móvel para consumo de relatórios
  - **Power BI Embedded** — Para incorporar visualizações em aplicações próprias
  - **Power BI Report Server** — Versão on-premises
- **Documentação:** <https://learn.microsoft.com/power-bi/>
- **Comunidade:** <https://community.powerbi.com/>
- **Treinamento gratuito (Microsoft Learn):** <https://learn.microsoft.com/training/powerplatform/power-bi>

## 📗 3. Microsoft Excel

**Descrição:** Amplamente utilizado para análise de dados devido à sua simplicidade e funcionalidade integrada. Embora seja primariamente uma planilha eletrônica, o Excel oferece recursos poderosos de análise de dados como Tabelas Dinâmicas (Pivot Tables), Power Query para ETL, gráficos avançados, análise estatística e, mais recentemente, integração com Python e IA (Copilot).

- **Desenvolvedor:** Microsoft
- **Plataforma:** Windows, macOS, Web, Mobile (iOS, Android)
- **Licença:** Comercial (incluso no Microsoft 365; versão web gratuita disponível)
- **Site oficial:** <https://www.microsoft.com/microsoft-365/excel>
- **Excel Online (gratuito):** <https://www.office.com/launch/excel>
- **Recursos de análise de dados:**
  - **Tabelas Dinâmicas (Pivot Tables)** — Resumo e agregação interativa de dados
  - **Power Query** — Importação, transformação e limpeza de dados (ETL)
  - **Power Pivot** — Modelagem de dados e cálculos DAX
  - **Gráficos e Mapas** — Visualizações integradas incluindo mapas geográficos
  - **Solver e Análise de Cenários** — Otimização e simulação
  - **Python no Excel** — Integração com Python para análise avançada
- **Documentação:** <https://support.microsoft.com/excel>
- **Treinamento gratuito (Microsoft Learn):** <https://learn.microsoft.com/training/excel>
