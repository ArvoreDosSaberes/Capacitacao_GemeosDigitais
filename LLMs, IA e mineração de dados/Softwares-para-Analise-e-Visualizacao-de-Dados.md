## 🔧 1. Cornerstone

**Descrição:** Focado em análise exploratória com ferramentas avançadas de modelagem. Oferece um ambiente completo para Análise Exploratória de Dados (EDA), construção de modelos e Design de Experimentos (DoE). Interface drag-and-drop com gráficos dinamicamente vinculados.

- **Desenvolvedor:** camLine GmbH (agora Elisa IndustriQ)
- **Plataforma:** Windows
- **Licença:** Comercial (versão de avaliação disponível)
- **Site oficial:** <https://www.camline.com/products/cornerstone>
- **Download / Trial:** <https://www.elisaindustriq.com/camline/products/cornerstone>

---

## 🌐 2. Data Applied

**Descrição:** Solução web para visualização e mineração de dados complexos. Implementa uma coleção de ferramentas de visualização e algoritmos para análise e mineração de dados, incluindo treemaps, séries temporais, detecção de outliers, árvores de decisão, regras de associação, clustering e mapas auto-organizáveis.

- **Desenvolvedor:** Data Applied (fundada por ex-funcionários da Microsoft)
- **Plataforma:** Web
- **Licença:** Comercial
- **Status:** ⚠️ **Software descontinuado** — o site original (dataapplied.com) não está mais disponível. A empresa encerrou suas operações.
- **Referência histórica:** <https://en.wikipedia.org/wiki/Data_Applied>

## 📈 3. GGobi

**Descrição:** Software livre para visualização interativa, ideal para dados multivariados. Ferramenta de manipulação direta para visualizar dados de alta dimensionalidade através de gráficos vinculados. A GUI é construída usando Gtk+. Possui integração com R através do pacote `rggobi`.

- **Desenvolvedor:** GGobi Foundation (AT&T Labs – Research)
- **Plataforma:** Windows, macOS, Linux
- **Licença:** Open Source
- **Site oficial:** <http://ggobi.org/>
- **Código-fonte (GitHub):** <https://github.com/ggobi/ggobi>
- **Documentação:** <http://ggobi.org/docs/>

> **Nota:** O GGobi original não recebe atualizações frequentes. O projeto evoluiu para o pacote R **GGally**, que estende o `ggplot2`:
>
> - **GGally (CRAN):** <https://cran.r-project.org/package=GGally>
>
> - **GGally (GitHub):** <https://github.com/ggobi/ggally>
>
> Para instalar no R:
>
> ```r
> install.packages("GGally")
> ```

## 📊 4. JMP

**Descrição:** Desenvolvido pelo SAS Institute, combina visualização dinâmica com análise estatística robusta. Oferece análise exploratória interativa, modelagem preditiva, design de experimentos e muito mais, tudo com interface gráfica intuitiva e sem necessidade de programação.

- **Desenvolvedor:** SAS Institute
- **Plataforma:** Windows, macOS
- **Licença:** Comercial (versão de avaliação gratuita de 30 dias)
- **Site oficial:** <https://www.jmp.com/>
- **Download / Trial gratuito:** <https://www.jmp.com/en/download-jmp-free-trial>
- **Versões disponíveis:**
  - **JMP** — Análise estatística e visualização
  - **JMP Pro** — Inclui modelagem preditiva avançada e machine learning
  - **JMP Clinical** — Voltado para indústria farmacêutica
  - **JMP Genomics** — Para análise genômica
