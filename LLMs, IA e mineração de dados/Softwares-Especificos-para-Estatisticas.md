## 📐 1. Minitab

**Descrição:** Focado em fornecer ferramentas precisas para análise estatística. Oferece controle estatístico de processos (CEP), planejamento de experimentos (DoE), análise de regressão, testes de hipóteses e gráficos estatísticos. Amplamente utilizado na indústria para projetos Six Sigma e melhoria contínua.

- **Desenvolvedor:** Minitab, LLC
- **Plataforma:** Windows, macOS, Web
- **Licença:** Comercial (versão de avaliação gratuita disponível)
- **Site oficial:** <https://www.minitab.com/>
- **Download / Trial gratuito:** <https://www.minitab.com/products/minitab/free-trial/>
- **Versões disponíveis:**
  - **Minitab Statistical Software** — Análise estatística completa
  - **Minitab Workspace** — Ferramentas visuais para melhoria de processos
  - **Minitab Connect** — Integração e preparação de dados

## 🧮 2. SPSS

**Descrição:** Popular entre pesquisadores para análise estatística complexa e mineração de dados. Oferece uma ampla gama de técnicas estatísticas, incluindo análise descritiva, regressão, ANOVA, análise fatorial, clustering e séries temporais. Interface gráfica intuitiva com suporte a sintaxe de comandos para automação.

- **Desenvolvedor:** IBM
- **Plataforma:** Windows, macOS, Linux
- **Licença:** Comercial (versão de avaliação gratuita de 30 dias)
- **Site oficial:** <https://www.ibm.com/spss>
- **Download / Trial gratuito:** <https://www.ibm.com/account/reg/signup?formid=urx-19774>
- **Versões disponíveis:**
  - **SPSS Statistics** — Análise estatística avançada
  - **SPSS Modeler** — Mineração de dados e modelagem preditiva
  - **SPSS Amos** — Modelagem de equações estruturais (SEM)

## 📈 3. Stata

**Descrição:** Combina análise de dados, visualização e modelagem estatística em uma única ferramenta. Amplamente utilizado em economia, ciências sociais, epidemiologia e biomedicina. Suporta regressão, análise de sobrevivência, dados em painel, séries temporais, modelos multinível e muito mais. Possui linguagem de programação própria para automação e extensão.

- **Desenvolvedor:** StataCorp LLC
- **Plataforma:** Windows, macOS, Linux
- **Licença:** Comercial (licenças anuais e perpétuas)
- **Site oficial:** <https://www.stata.com/>
- **Download / Trial gratuito:** <https://www.stata.com/customer-service/evaluate-stata/>
- **Versões disponíveis:**
  - **Stata/BE** (Basic Edition) — Para datasets de tamanho moderado
  - **Stata/SE** (Standard Edition) — Para datasets grandes
  - **Stata/MP** (Multiprocessor) — Para grandes datasets com processamento paralelo
